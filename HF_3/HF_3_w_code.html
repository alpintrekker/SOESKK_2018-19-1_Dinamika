II. Pályaegyenlet¶

Bevezetve a $\gamma=e^{0.1t}$ segédváltozót, a mozgásegyenletek: $$x = f(t) = 3e^{0.1t} = 3\gamma\label{iksz}$$ és $$y=\varphi(t)=10e^{-0.1t} = \frac{10}{\gamma}\label{ipsz}$$ behelyettesítéssel kapjuk, hogy: $$y = \frac{30}{x}$$
